QUSAR Interserver Communications Protocol (qICP)

What is QICP?

qICP is a communications protocol designed by QUASAR Systems to allow text relay between Discord servers. It is used by QUASAR Chat to allow for communications between servers, and can also be modified for private use.

How it works

qICP works by using a bot to scan messages in a server, and waiting until a message is sent by a user that meets the criteria for being relayed. Once a message is relayed, the bot will send the message to the other server(s) that are connected to it, and the message will be sent in a channel that is designated for relayed messages. The bot will also scan for messages in the designated channels, and relay them back to the original server if they meet the criteria for being relayed.

The criteria for being relayed can be set up by the user, and can include things like routing tags, user roles and more. This allows for ultimate customizability and integration with existing server setups.

The system uses the Discord webhook system, allowing for messages to be sent with information from the original sender, which can be either the server name or the original sender's username, depending on the user's preferences. This allows for a more seamless experience, and allows for users to easily identify where messages are coming from.

Furthermore, the usage of webhooks allows for messages to be sent at faster speeds than traditional bot messages, allowing for a more seamless experience. This is especially important for larger servers, where message volume can be high and speed is of the essence.

Use Cases

qICP can not only be used for chat apps, but can be modified to be used for a variety of other purposes, such as:

    - Custom notification systems

    qICP can be used to create custom interserver notification systems, such as emergency alert networks.

    - Internal communication tools

    qICP can be used to facilitate communication between different Discord servers within an organization, without being exposed to the risks of direct QUASAR Chat usage.

    - And so much more!
