QUASAR Chat

QUASAR Chat is a revolutionary Discord message relay system utilising efficient architecture to facilitate messange exchange between Discord servers.

Key Features

- Seamless Message Relay: Effortlessly relay messages between Discord servers, ensuring smooth communication across communities.
- Efficient Architecture: Built on the native Discord webhook system, QUASAR Chat is able to relay messages between servers in miliseconds.
- User-Friendly Interface: Intuitive interface design allows users to easily set up and manage their message relay configurations.
- Privacy First Design: Utilising the 3WE encryption stamdard. messages and user data are protected with multi-layer end-to-end encryption.

Technical Explanation

QUASAR Chat utilises a central exchange server, similar to a PBX. A bot scans all messages in a server it is installed, and when an authorised role sends a message with a valid tag, it gets routed to the Central Exchange Server. The Central Exchange Server pulls its data from a dictionary file, similar to a phonebook.


The Phonebook file, internally known as webhooks.json sorts all information per server. This file is encrypted, and as such in the event of data theft, the thief will not have anything useful left. The data is entered by high-level QUASAR administrators, and is deleted from the Discord enviorenment after it is entered.


The Central Exchange Server is responsible for routing the messages, censoring them, preventing flooding, DDoS attacks and other platform abuse. It's the brain of the program.
